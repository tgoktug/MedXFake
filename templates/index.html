MedXFake

A New, Comprehensive, Realistic and Applicable Deep-Fake Synthesis, Manipulation and Detection System for Medical Images

Abstract

Accessing the outputs of medical imaging applications is costly and challenging due to factors such as protecting patient rights and confidentiality of personal data. Image synthesis methods developed to solve these problems are dangerous in terms of medical data security.

In this study, deepfake images were generated using DDPM and GAN models on medical images collected from online platforms. A realistic medical image manipulation pipeline was built using a DDPM-based method. Furthermore, a large medical deepfake dataset was created, and different learning strategies were implemented for deepfake detection.

Fake images synthesized from scratch were detected using a ResNet-based classifier, achieving an F1-score of 99.783%. Detecting manipulated images is more challenging; therefore, a contrastive learning–based fine-tuning stage was applied, boosting CNN detection accuracy from 89.9% to 98%.

All synthesis, manipulation, and detection modules were integrated into a unified API system, and a user-friendly web application was developed to bring high-performance medical deepfake generation and detection capabilities to the end user.

Available Modules

DDPM Manipulation

Upload an image, select a bounding box, and apply DDPM-based medical image manipulation.

Manipulation Detection

Binary classifier for detecting DDPM-based medical image manipulation.

8-Class Fake Detection

Identify real vs synthetic brain, lung, kidney and breast image pairs using a ResNet model.

Image Synthesis

Generate medical deepfake images (Brain MRI, Kidney CT, Lung CT, Breast USG) using DDPM models.
